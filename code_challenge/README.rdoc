== README

How to get this app up and running:

* Ruby version - 2.2.0

* Clone this repository

* Run `bundle` inside the directory to install gems

* Get Postgres running. Version 9.4.4

* Run `rake db:create` and `rake db:migrate`

* Start your rails server `rails s`

* Open application at localhost:3000 in web browser to start uploading some CSV!!
